Codingame Atecna

Mode de jeu : Le + rapide

Timer:

Problème

Vous recevez 2 entiers A et B en entrée. Le résultat doit être la somme des 2 entiers.

Entrées:

input = [ A, B ]

Sortie:

Retourne un entier qui est la somme de A et B

Contraintes:

  • -500 000 <= A <= 500 000
  • -500 000 <= B <= 500 000

Exemples:

Entrée

3 0

Sortie

3

0

Autres joueurs

Sortie standard:

> 11122
Success
  1. Test 1 : 1 + 2
  2. Test 2 : 5 + 0
